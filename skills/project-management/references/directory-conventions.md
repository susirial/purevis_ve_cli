# 项目资产目录规范

所有项目资产文件必须且只能保存在 `output/projects/<项目名称>/` 路径下。

## 目录结构

```
output/projects/<项目名称>/
├── subjects/                    # 角色/场景/物品等主体库设定与参考图
├── episodes/
│   └── <剧集ID>/               # 如 ep01, ep02
│       ├── scripts/            # 剧本文档与分镜描述
│       ├── storyboard/         # 分镜文件
│       ├── keyframes/          # AI 视频关键帧图片
│       └── video/              # 最终视频片段
└── state.json                  # 项目状态文件
```

## 命名规范

- 剧集 ID：严格使用 `ep01`, `ep02` 格式，不得使用 `S01` 或中文
- 主体命名：`主体_<类型>_<名称>_<变体描述>`（如 `主体_人物_崔秀妍_穿短裙.jpg`）
- 禁止在工作目录中创建不规范的中英文文件夹
