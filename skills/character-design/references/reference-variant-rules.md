# 角色参考图变体详细规则

## pure_character（纯人物参考图）

画面要求：
- 单人、单主体、全身完整展示
- 9:16 竖构图、纯白背景、studio lighting
- 单张图中只出现一个角色
- 只允许一个默认站姿或一个轻微转身展示姿态

禁止项：
- 多视图、角色设定拼板、turnaround sheet
- pose sheet、expression sheet、collage
- 同一角色多个不同站姿
- 武器、坐骑、宠物、伴生体、额外角色
- 额外手持道具、环境叙事元素

## full_character（完整角色设定图）

画面要求：
- 单角色、单主体、9:16 竖构图、纯白背景
- 只允许一个站姿或一个默认展示姿态
- 允许保留角色的标志性武器、手持道具、挂件或身份强绑定 props

禁止项：
- 多视图、多人拼贴、角色 lineup
- pose sheet、expression sheet
- 坐骑或额外角色（除非用户明确要求）

## mounted_character（带坐骑设定图）

画面要求：
- 9:16 竖构图、纯白背景
- 以单个主角色为中心
- 只允许出现明确指定的坐骑、伴生物或标志性武器

禁止项：
- 额外角色、crowd
- 多视图、拼贴
- 多个不同站姿版本同时出现在一张图内
