# 审美评审标准参考

## 角色参考图评审
- 是否单人单主体
- 是否纯白背景
- 是否符合指定的 reference_variant（pure_character / full_character / mounted_character）
- 服装、发型、体态是否与文字设定一致
- 是否出现了不应出现的元素（武器、坐骑、额外角色等）

## 场景图评审
- 是否真正的空镜头（无人物/动物）
- 环境元素是否与设定一致
- 光影和氛围是否合理

## 关键帧图评审
- 是否与分镜描述匹配
- 角色是否与参考图一致
- 构图是否符合指定的构图技法
- 光影是否符合指定的用光方案
- 前后帧是否保持连贯性

## 风格一致性
- 是否符合项目的结构化风格配置
- 色调、材质、渲染风格是否统一
