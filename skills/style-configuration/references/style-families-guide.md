# 风格家族与子类参考指南

本文档提供风格系统的分层结构说明，帮助在推荐风格时快速定位。

## 风格家族分层逻辑

风格系统采用两级分类：
- **Family（风格家族）**：顶层大类，如"二次元"、"写实"、"水墨"
- **Subtype（二级子类）**：家族内的细分变体，如"赛璐珞"、"厚涂"

## 适配矩阵

不同风格家族对不同资产类型（image / keyframe / video）的适配程度不同：
- 部分风格在静态图片中效果很好，但在视频中可能风格一致性难以维持
- 推荐时应说明该风格在不同资产类型中的表现预期

## 典型使用场景

| 用户意图 | 推荐方向 |
|---------|---------|
| "像新海诚" | 二次元 → 新海诚风 |
| "偏写实" | 写实 → 影视写实 |
| "水墨风" | 国风 → 水墨 |
| "赛博朋克" | 科幻 → 赛博朋克 |

## 注意事项

- 风格预设中的元数据（色板、镜头、材质）是推荐值，非硬约束
- 用户可以在预设基础上自定义调整
- 风格配置支持版本管理，可随时回滚
